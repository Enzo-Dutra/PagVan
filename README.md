🚌 Sistema de Gestão para Transporte Escolar
Projeto desenvolvido em HTML, CSS, PHP e SQL com o objetivo de criar um sistema web (não apenas um site) que auxilie motoristas de vans escolares a modernizar e facilitar a organização do seu trabalho, proporcionando maior eficiência, controle e qualidade no serviço.

🎯 Objetivo Geral
Desenvolver um sistema para uma motorista de van escolar com funcionalidades que melhorem a produtividade e permitam uma gestão eficiente de alunos, pagamentos e rotas escolares.


📊 Área de Controle / Planilha Mestre
Visualização de alunos por escola e por período

Ida e volta (chamada e conferência)

Situação de pagamento destacada por cor:

🟥 Vermelho: Em atraso

🟨 Amarelo: À vencer

🟩 Verde: Em dia

Hibernação dos cadastros para não perder dados
